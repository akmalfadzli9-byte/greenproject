# Greenhouse SaaS – Konteks Projek

## 📌 Nama Projek
**Greenhouse SaaS** – Sistem Pengurusan Ladang Rumah Hijau Pintar

## 🎯 Tujuan
Membantu pengurus ladang (umur 30-60, kurang celik IT) mengurus operasi rumah hijau secara digital merangkumi:

- Pemantauan sensor (suhu, kelembapan, pH, cahaya)
- Ramalan cuaca & cadangan ladang
- Pengurusan jualan hasil tanaman
- Inventori baja, benih, alatan
- Produktiviti pekerja & tugasan harian
- Laporan & analisis

## 🛠 Teknologi & Stack
| Komponen | Teknologi |
|----------|-----------|
| Markup | HTML5 Semantic |
| Styling | Tailwind CSS (CDN) + custom CSS |
| JavaScript | Vanilla ES6 |
| Charting | Chart.js v4.4.0 |
| Storage | localStorage (tiada backend) |
| Dark mode | `class` strategy dengan localStorage |
